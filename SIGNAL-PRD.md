# SIGNAL — Product Requirements Document (PRD v1.0)

**문서 유형:** 제품 요구사항  
**앱 이름:** SIGNAL  
**슬로건:** 노이즈는 걸러내고, 진짜 시그널만

---

## 1. 제품 개요

| 항목 | 내용 |
|------|------|
| **앱 이름** | SIGNAL |
| **슬로건** | 노이즈는 걸러내고, 진짜 시그널만 |
| **카테고리** | 금융 / 투자 정보 |
| **타깃** | 미국 주식에 관심 있는 한국 개인 투자자 (20~40대) |
| **플랫폼** | iOS, Android (React Native) |
| **비즈니스 모델** | 광고 (AdMob) + 프리미엄 구독 (광고 제거) |
| **목표 출시일** | MVP: 2개월 내 |

### 핵심 문제 정의

미국 투자에 관심 있는 한국 투자자들은 Bloomberg, Reuters 등 영문 정보를 실시간으로 소화하기 어렵고, 기존 국내 앱은 정보 과부하이거나 미국 시장에 최적화되어 있지 않다. **SIGNAL**은 AI가 핵심만 걸러내어 한국어로 즉시 전달한다.

## 2. 기능 명세

### MVP · Phase 1 (1~2개월)

| 기능 | 우선순위 | 설명 | API |
|------|----------|------|-----|
| **실시간 뉴스 피드** | 필수 | Finnhub / Marketaux API로 글로벌 속보 수집, Claude API로 한국어 3줄 요약 자동 생성 | Finnhub + Claude |
| **컨콜 요약** | 필수 | 분기 실적 발표 트랜스크립트 자동 수집 → CEO/CFO 핵심 발언, 가이던스, 리스크 자동 추출 및 요약 | API Ninjas + Claude |
| **투자 캘린더** | 필수 | 실적 발표 일정 · FOMC · 경제지표 한눈에 확인, 관심 종목 실적 발표 전날 자동 알림 | Finnhub Earnings Calendar |
| **맞춤 알림 (FCM)** | 필수 | 관심 키워드·종목 등록 시 관련 속보 즉시 푸시, 실적 발표 30분 전 자동 알림 | Firebase FCM |

### v2 · Phase 2 (3~4개월)

| 기능 | 우선순위 | 설명 | API |
|------|----------|------|-----|
| **AI 모닝 브리핑** | 중요 | 매일 오전 7시, 간밤 미국 시장 핵심 3가지를 Claude가 자동 생성하여 발송 | Claude + FCM |
| **종목별 뉴스 필터** | 중요 | 관심 종목 설정 시 해당 종목 관련 뉴스만 필터링하여 피드 구성 | Finnhub |
| **컨콜 하이라이트** | 중요 | 트랜스크립트에서 핵심 발언 자동 추출, 감성 점수(-1~+1) 표시 | Claude |

### v3 · Phase 3 (5~6개월)

| 기능 | 우선순위 | 설명 | API |
|------|----------|------|-----|
| **프리미엄 구독** | 선택 | 무제한 컨콜 요약, 모닝 브리핑, 고급 알림 (월 4,900원) | Stripe / 인앱결제 |
| **포트폴리오 연동** | 선택 | 내 보유 종목 기반 개인화 뉴스 피드 자동 구성 | 자체 개발 |
| **리서치 리포트 요약** | 선택 | 국내외 증권사 리포트 PDF → 핵심 요약 | Claude |

## 3. UX 정책

| 원칙 | 설명 |
|------|------|
| **정보 최소화** | 탭 5개 이하, 카드 한 화면에 3~4개. 숫자와 수치는 핵심만. |
| **속도 우선** | 뉴스 피드 초기 로딩 2초 이내. 스켈레톤 UI로 체감 속도 개선. |
| **다크 모드 기본** | 미국 장 시간대(새벽)에 사용하는 앱 특성상 다크 모드를 기본으로. |
| **알림 품질 관리** | 하루 최대 10건. 중요도 HIGH만 즉시 발송, 나머지는 배치. |
| **AI 투명성** | Claude 요약임을 명시. '원문 보기' 링크 항상 제공. |
| **한국어 우선** | 모든 뉴스 요약은 한국어. 종목명은 영문 병기 (엔비디아 NVDA). |

## 4. 기술 스택

| 레이어 | 스택 | 선정 이유 |
|--------|------|-----------|
| 앱 프레임워크 | React Native (Expo) | iOS/Android 동시 개발, OTA 업데이트 |
| 뉴스 API | Finnhub + Marketaux | 무료 플랜으로 MVP 시작, 유료 확장 용이 |
| 컨콜 트랜스크립트 | API Ninjas / FMP | 2005년부터 히스토리, 감성 점수 내장 |
| AI 요약 | Claude API (Sonnet 4.6) | 한국어 요약 품질 최고, 컨텍스트 200K |
| 푸시 알림 | Firebase FCM | 무료, 설정 간단, 세그먼트 타깃 지원 |
| DB / 실시간 | Firebase Firestore | 실시간 뉴스 동기화, 무료 티어 넉넉 |
| 인증 | Firebase Auth | 소셜 로그인 (카카오·Apple) 연동 |
| 결제 (v3) | RevenueCat | iOS/Android 인앱결제 통합 관리 |

## 6. 수익화 전략

### 핵심 전략

무료 사용자에게 AdMob 네이티브 광고를 노출하고, 광고가 불편한 헤비 유저가 자연스럽게 프리미엄으로 전환하는 구조. 광고 품질과 UX를 동시에 유지하는 것이 핵심.

### 플랜 비교

**무료 플랜**

- 뉴스 피드: 뉴스 5개마다 네이티브 광고 1개 삽입
- 컨콜 요약: 월 10건 제한
- 모닝 브리핑: 광고 포함 발송
- 알림: 하루 최대 5건

**프리미엄 (월 4,900원)**

- 광고 완전 제거
- 컨콜 요약 무제한
- 모닝 브리핑 광고 없이 수신
- 알림 하루 최대 20건
- 프리마켓 속보 우선 수신

### AdMob 광고 유형

| 유형 | 위치 | 우선순위 | CPM (참고) | 이유 |
|------|------|----------|--------------|------|
| 네이티브 광고 | 뉴스 피드 (5개마다 1개) | 메인 | ₩500~₩1,500 | 뉴스 카드 형태와 자연스럽게 혼합 |
| 배너 광고 | 캘린더 · 관심종목 탭 하단 | 보조 | ₩100~₩400 | 정보 탐색 중 자연스러운 노출 |
| 전면 광고 | 앱 첫 실행 시 (하루 1회) | 선택 | ₩1,000~₩3,000 | 수익 극대화, 빈도 엄격 제한 |

### 월 수익 추정 (ARPDAU 기준, 프리미엄 전환 5% 포함 추정치)

| 시나리오 | DAU | ARPDAU | 월 수익 |
|----------|-----|--------|---------|
| 보수적 | 1,000명 | ₩8 | ₩240,000 |
| 중간 | 3,000명 | ₩10 | ₩900,000 |
| 낙관적 | 10,000명 | ₩12 | ₩3,600,000 |

### 운영 정책 (AdMob)

- 광고는 뉴스 카드와 시각적으로 구분 ("광고" 라벨 필수 — 앱스토어 정책)
- 금융 사기·가상화폐 투자 광고 카테고리 차단 설정
- 광고 로딩 실패 시 빈 공간 없이 뉴스 카드로 자연 대체
- 프리미엄 사용자에게는 광고 SDK 자체를 초기화하지 않음 (배터리·데이터 절약)

## 7. 리스크 & 대응

| 리스크 | 영향도 | 대응 |
|--------|--------|------|
| API 비용 급증 | HIGH | Claude 요약 캐싱 (동일 기사 재요약 방지), 프리 플랜 한도 모니터링 |
| 뉴스 요약 오류 | MED | 'AI 요약' 라벨 + 원문 링크 항상 제공, 법적 면책 문구 삽입 |
| 앱스토어 심사 거절 | MED | 금융 정보 앱 가이드라인 사전 검토, 투자 권유가 아닌 '정보 제공' 명시 |
| 알림 피로도 | MED | 하루 최대 알림 수 설정, 중요도 필터링, 사용자 직접 조정 가능 |
| 경쟁사 유사 기능 출시 | LOW | 컨콜 요약 + AI 큐레이션 조합의 빠른 실행으로 선점 |
